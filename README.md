# advent-of-code

[Advent of code website](https://adventofcode.com/)

### 2022 (js)
Install node
```sh
npm i -g node
```

run via 
```sh
node ./2022/day1.mjs
```

### 2023+ (ts)
Install ts-node
```sh
npm i -g ts-node
```

Create boilerplate for new day
```sh
sh new_day.sh <day> <year>
```

run via 
```sh
ts-node ./2023/day1
```
